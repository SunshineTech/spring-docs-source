= 填充器命名空间参考

[[namespace-dao-config]]
== <populator />元素
`<populator />` 元素允许通过Spring Data存储库基础结构填充数据存储。footnote:[请参阅<<repositories.create-instances.spring>>]

.属性
[options="header", cols="1,3"]
|===============
|名称|描述
|`locations`|找到要从中读取填充存储库的对象的文件位置。
|===============
